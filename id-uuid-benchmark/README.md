# ID UUID Benchmark

A simple Go project for benchmarking ID and UUID operations.

## Project Structure

- `main.go`: Entry point for the application
- `pkg/`: Directory for reusable packages

## Getting Started

### Prerequisites

- Go 1.16 or higher

### Building the Project

```bash
go build
```

### Running the Project

```bash
go run main.go
```

Or after building:

```bash
./id-uuid-benchmark
```

## Adding Dependencies
